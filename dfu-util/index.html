Kiwrious Sensor Update Utility

For UV and Heart Rate Sensors
  1. Click Step 1 button and select the Kiwrious Sensor, sensor light should be flashing now
  2. Click Step 2 button and select the Kiwrious DFU
  3. Click Step 3, check Status below
  4. If you see "Done!", then the sensor update is complete. Please unplug the sensor and plug it back in to use it in https://learn.kiwrious.com/ or https://measure.kiwrious.com/ Otherwise, please contact us if any issues at support@kiwrious.com
